Gesture Action
1.Pinch 1.Start changing some value. When pinching, then twisting the hand, you can adjust for instance the volume of an instrument
2.Thumbs up 2.Turn on instrument
3.Thumbs down 3.Turn off instrument
4.Closed fist 4.Intensity up
5.Open palm 5.Intensity down
6.V-sign 6.Play a soothing track
7.Pointing in a direction

7.Switch track:

  1. Up → first
  2. Down → last
  3. Right → next
  4. Left → previous
